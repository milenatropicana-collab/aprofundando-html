Um ponto sobre HTML e CSS

HTML

HTML é a linguagem padrão de marcação para criar a estrutura de páginas web. Ela utiliza tags para organizar conteúdo — como títulos, parágrafos, links, imagens e tabelas. O HTML não é uma linguagem de programação, mas sim a base estrutural (esqueleto) visualizada por navegadores, sendo frequentemente complementada por CSS (estilo) e JavaScript (funcionalidade).

Segundo a documentação do mozilla

No MDN Web Docs (Mozilla), existem mais de 100 elementos HTML, com algumas fontes baseadas na documentação indicando cerca de 142 tags no total, incluindo elementos modernos e obsoletos. A lista abrange diversos propósitos, desde estrutura básica até multimídia, formulários e semântica, cobrindo todo o padrão HTML5 e anteriores.

As principais tags HTML formam a estrutura base html, head, body e a organização de conteúdo h1-h6, p, a, img de uma página web. Tags semânticas modernas como header, nav, section, article e footer são cruciais para SEO e acessibilidade, substituindo o uso excessivo de div.

CSS

CSS é uma linguagem de estilos usada para definir a apresentação visual de documentos HTML. Ela separa o conteúdo (estrutura) da forma (design), permitindo estilizar cores, fontes, layouts e criar sites responsivos. As regras CSS funcionam em cascata, aplicadas via seletores a elementos HTML, facilitando a manutenção e consistência.
